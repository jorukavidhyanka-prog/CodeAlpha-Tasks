# AI Chatbot in Java

## Overview
This project is a simple AI Chatbot developed in Java. The chatbot interacts with users through a console interface and responds to predefined questions and commands.

## Features
- Interactive chat interface
- Responds to common greetings
- Answers basic questions
- Simple rule-based conversation
- Easy to customize and extend

## Technologies Used
- Java
- Scanner Class
- Conditional Statements
- Loops

## Project Structure

```text
AIChatBot.java
```

## How It Works
1. The user enters a message.
2. The chatbot analyzes the input.
3. Based on predefined rules, the chatbot generates a response.
4. The conversation continues until the user types `bye`.

## Running the Application

### Compile

```bash
javac AIChatBot.java
```

### Run

```bash
java AIChatBot
```

## Example Conversation

```text
You: Hello
Bot: Hi! How can I help you?

You: What is your name?
Bot: I am a Java AI Chatbot.

You: How are you?
Bot: I am doing great. Thanks for asking!

You: bye
Bot: Goodbye! Have a nice day.
```
